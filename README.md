### Wikipedia Android app

 Проверка перехода в браузер
  1. Открываем приложение
  2. Нажимаем на кнопку "Еще"
  3. Нажимаем кнопку "Пожертвовать"
  4. Проверяем, что случился переход на окно выбора браузера для открытия
